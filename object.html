Description of the NL_2D_FEM object¶

The finite element (FE) information and methods are defined in a Matlab object called NL_2D_FEM. An instance of the FE model (i.e. a new FE model) can be created using the default constructor: model = NL_2D_FEM. This creates an object model containing the data of a given problem and methods to perform the data analysis.

The attributes and methods of the NL_2D_FEM class (denoted model) are given below:

Attributes¶

begin{enumerate}

item model.geom: The geometry of the model (key points and lines of the geometry) begin{enumerate}

item model.geom.geom_node: List of number and coordinates of the key points of the geometry. Each line is of the type point number, x coordinate, y coordinate’. item :mod:`model.geom.geom_element: List of lines in the geometry connecting the nodes. Each line is of the type line number, point 1, point 2’. item :mod:`model.geom.discretisation: List of number of elements in each line (list of integers, with size the number of lines of the geometry).

end{enumerate}

item model.prop: Element cross-section and material properties (constant properties for all elements of the model) begin{enumerate}

item model.prop.area: Area of the cross-section. item model.prop.inertia: Second moment of area of the cross-section. item model.prop.shear_coeff_k: Shear coefficient of the cross-section. item model.prop.density: Material density. item model.prop.young_mod: Material Young’s modulus. item model.prop.shear_mod: Material shear modulus. item model.prop.poisson: Material Poisson’s ratio. item model.prop.alpha: Material damping coefficient (linear Rayleigh damping).

end{enumerate}

item model.mesh: mesh of the FE model begin{enumerate}

item model.mesh.nodes: List of number and coordinates of the nodes of the mesh. Each line is of the type node number, x coordinate, y coordinate’. item :mod:`model.mesh.connect: Connectivity table, textit{i.e.} list of elements of the mesh. Each line is of the type element number, node 1, node 2’. item :mod:`model.mesh.number_nodes: Total number of nodes in the mesh. item model.mesh.number_elements: Total number of elements in the mesh.

end{enumerate}

item model.visu: Visualization information for displaying results begin{enumerate}

item model.visu.visu_node_list: List (cell array) of observed node and degree of freedom (dof). begin{itemize}

item model.visu.visu_node_list{n} is a structure with two fields: begin{enumerate}

item mds.prop.visu.visu_node_list{n}.node: Node number. item mds.prop.visu.visu_node_list{n}.dof: List of dof number(s).

end{enumerate}

end{itemize} item model.visu.dof: List of indices of observed dof in the assembled vector.

end{enumerate}

item model.boundary: Boundary conditions information begin{enumerate}

item model.boundary.bc_node_list: List (cell array) of boundary condition nodes and dof. begin{itemize}

item mds.boundary.bc_node_list{n} is a structure with two fields: begin{enumerate}

item mds.boundary.bc_node{n}.node: Node number. item mds.boundary.bc_node{n}.dof: List of dof number(s).

end{enumerate}

end{itemize} item model.boundary.dof_list: List of indices of all dof in the assembled vector. item model.boundary.prescribed_dof: List of indices of all prescribed dof (displacement or rotation set to zero). item model.boundary.active_dof: List of indices of all non-boundary condition dof.

end{enumerate}

item model.loads: Loads (static and periodic, point force and distributed loads) begin{enumerate}

item model.loads.static: Static loads. begin{enumerate}

item model.loads.static.static_ponctual_force_node_list: List (cell array) of forced node(s) and dof. begin{itemize}

item model.loads..static.static_ponctual_force_node_list{n} is a structure with three fields: begin{enumerate}

item model.loads..static.static_ponctual_force_node_list{n}.node: Node number. item model.loads..static.static_ponctual_force_node_list{n}.dof: List of dof number(s). item model.loads.static.static_ponctual_force_node_list{n}.amplitude: List of force amplitudes (one amplitude for each dof in the previous field).

end{enumerate}

end{itemize} item model.loads.static.static_distributed_force_direction: List of directions among (1) x-axis (2) y-axis or (3) moment. item model.loads.static.static_distributed_force_amplitude: List of amplitudes (linear force density).

end{enumerate} item model.loads.periodic: Periodic loads

begin{enumerate}

item model.loads.periodic.periodic_ponctual_force_node_list: List (cell array) of forced node(s) and dof. begin{itemize}

item model.loads.periodic.periodic_ponctual_force_node_list{n} is a structure with three fields: begin{enumerate}

item model.loads.periodic.periodicponctual_force_node_list{n}.node: Node number. item model.loads.periodic.periodic_ponctual_force_node_list{n}.dof: List of dof number(s). item model.loads.periodic.periodic_ponctual_force_node_list{n}.amplitude: List of force amplitudes (one amplitude for each dof in the previous field).

end{enumerate}

end{itemize} item model.loads.periodic.periodic_distributed_force_direction: List of direction among (1) x-axis (2) y-axis or (3) moment. item model.loads.periodic.periodic_distributed_force_amplitude: List of amplitudes (linear force density).

end{enumerate}

end{enumerate}

item model.matrices: Assembled matrices of the model begin{enumerate}

item model.matrices.mass: Mass matrix. item model.matrices.damping: Damping matrix. item model.matrices.stiffness_at_origin: Linear stiffness matrix (around undeformed (reference) configuration). item model.matrices.tangent_stiffness_at_qs: Unused.

end{enumerate} item model.vectors: Assembled vectors of the model begin{enumerate}

item model.vectors.null_vector: Assembled vector of dof full of zeros. item model.vectors.static_forces: Assembled vector of static forces. item model.vectors.periodic_forces: Assembled vector (matrix) of periodic forces (each row corresponds to a harmonic). item model.vectors.static_solution: Unused.

end{enumerate}

item model.solver: Solver information begin{enumerate}

item model.solver.type: 'homemade': Default, solve NL system with a simple Newton method, or 'fsolve': solve NL system with Matlab fsolve from optimization toolbox). item model.solver.tol_res: Residual tolerance. item model.solver.tol_step: Step tolerance. item model.solver.n_iter_max: Number of maximum iterations for Newton’s method.

end{enumerate}

end{enumerate}

Methods¶

Initialization methods¶

begin{enumerate}

item model.set_model_from_mds: Set the model attributes from a model data structure mds. item model.set_geom: Set the geometry of the model (key points and lines of the geometry). item model.set_prop: Set the cross-section and material properties. item model.set_boundary: Set the boundary conditions. item model.set_visu: Set the visualization information for displaying result. item model.set_static_loads: Set the static loads (point force and distributed loads). item model.set_periodic_loads: Set the periodic loads (point force and distributed loads). item model.set_mesh: Set mesh from user inputs.

end{enumerate}

FEM methods¶

paragraph{Mesh-related methods}

begin{enumerate}

item model.auto_mesh: Create a mesh from the geometry information. item model.mesh_from_truss: Used in auto_mesh.

end{enumerate}

paragraph{Element-related methods}

begin{enumerate}

item model.elementOrientation: Compute element length and orientation angle. item model.transformationMatrix: Compute rotation matrix from an orientation angle. item model.elementary_mass_matrix: Elementary mass matrix of the Timoshenko beam model. item model.elementary_tangent_stiffness_matrix: Elementary tangent stiffness matrix from a given (elementary) dof vector. item model.elementary_internal_force_vector: Elementary vector of nonlinear internal forces from a given (elementary) dof vector.

end{enumerate}

paragraph{Assembly-related methods}

begin{enumerate}

item model.assemble_constant_matrix: Assemble a matrix depending on the input (textit{e.g.} mass, stiffness). item model.assemble_constant_vector: Assemble a vector depending on the input (textit{e.g.} [static, periodic, internal] forces). item model.initialise_matrices_and_vector: Assemble the mass and stiffness matrices and the static and periodic force vectors.

end{enumerate}

Mechanics-related methods¶

begin{enumerate}

item model.strains_and_stress_at_gauss_point: Return stains, stress and gradient information from a given dof vector. item model.solve_static_problem: Solve the static problem. item model.linear_modal_analysis: Linear modal analysis around a given configuration. item model.buckling_analysis: Simple buckling analysis. item model.linear_analysis: Linear forced response.

end{enumerate}

MAN-related methods¶

begin{enumerate}

item model.set_MAN_parameters: Set the parameters structure. item model.initialise_MAN_computation: Automatic initialization of a MAN computation. Performs a static, modal anlysis around a static configuration and computes a MAN starting point (U0) depending on the type of computation. item model.solve_MAN_system_at_fixed_amplitude: Solve the MAN system such that the first harmonic of a given dof is fixed (variable frequency); used in the initialization method. item model.solve_MAN_system_at_fixed_frequency: Solve the MAN system such that the frequency is fixed (variable amplitude); used in the initialization method. item model.man_initial_point: Returns a MAN point from a given static and periodic vector of dof. item model.man_auxiliary_variable_vector: Computes the MAN auxiliary variables vector from a given vector of dof. item model.auxiliary_variable_expression: Used in the MAN equation to define the auxiliary variables.

end{enumerate}

Equation (residual)-related methods¶

begin{enumerate}

item model.static_residual: Used to solve the static problem. item model.newton_solver: Homemade Newton solver used to solve the static problem or the MAN initial point (or use the Matlab function fsolve from the optimization toolbox).

end{enumerate}

Display-related methods¶

begin{enumerate}

item model.plot_deformed_mesh: Plot deformed structure from a given vector. item model.plot_static_configuration: Plot the static configuration. item model.plot_mode_shape: Plot the mode shapes from a given list.

end{enumerate}
